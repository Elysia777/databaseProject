🔧 退款订单修复验证测试

问题描述

在添加投诉退款功能后，乘客无需支付也能继续发起下一个订单，订单页面也没有显示要求乘客支付。

根本原因

  • 后端逻辑错误：未支付订单检查函数没有排除 REFUNDED 状态的订单
  • 前端逻辑正确：前端的 isUnpaid 函数和 hasUnpaidOrders 状态管理都是正确的

修复内容

// 修复前（OrderController.java） boolean hasUnpaid = orders.stream() .anyMatch(order -> "COMPLETED".equals(order.getStatus()) && "UNPAID".equals(order.getPaymentStatus())); // 修复后 boolean hasUnpaid = orders.stream() .anyMatch(order -> "COMPLETED".equals(order.getStatus()) && "UNPAID".equals(order.getPaymentStatus()) && !"REFUNDED".equals(order.getPaymentStatus()));
