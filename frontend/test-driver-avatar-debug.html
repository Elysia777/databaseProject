📊 头像数据保存机制说明

1. 数据库存储结构： - users表中存储头像URL: avatar VARCHAR(255) - 存储相对路径，如：/uploads/avatars/uuid-filename.jpg - 文件系统存储在: uploads/avatars/目录 2. 头像上传流程： 1. 前端上传图片 → 后端FileUploadController 2. 文件保存到 uploads/avatars/ 目录 3. 生成相对路径URL：/uploads/avatars/filename.jpg 4. 更新users表的avatar字段 5. 前端访问时加上服务器前缀：http://localhost:8080/uploads/avatars/filename.jpg 3. 司机信息传递流程： 1. OrderDispatchService获取Driver信息 2. 通过driver.getUserId()获取User信息 3. WebSocketNotificationService构建司机信息，包含user.getAvatar() 4. 通过WebSocket发送给乘客端 5. 乘客端PassengerMap.vue接收并显示
