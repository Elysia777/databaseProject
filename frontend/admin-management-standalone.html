🛡️ 管理员账号管理 (独立版)

由于CORS限制，此页面提供SQL脚本方式创建管理员

⚠️ CORS跨域问题说明

直接打开HTML文件会遇到CORS跨域问题，无法直接调用后端API。

解决方案：

1. 启动HTTP服务器访问页面

2. 使用下面提供的SQL脚本直接操作数据库

🚀 启动HTTP服务器

在frontend目录下运行以下命令之一：

方案1 - 使用项目自带服务器：
cd frontend
node start-test-server.js
# 然后访问: http://localhost:3000/admin-management.html

方案2 - 使用Python：
cd frontend
python -m http.server 3000

方案3 - 使用Node.js：
cd frontend
npx http-server -p 3000

查看现有管理员

-- 查看所有管理员用户 SELECT u.id, u.username, u.real_name, u.phone, u.email, u.user_type, u.status, a.role, a.department, a.position, a.is_active FROM users u LEFT JOIN admin_users a ON u.id = a.user_id WHERE u.user_type = 'ADMIN';
